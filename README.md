# 贞观勇士（Zhenguan Warriors）

> 一款致敬《曹操传》的回合制战棋 RPG 手游，主人公为李世民，讲述其从晋阳起兵到登基为帝的征战史诗。

## 项目状态

- **当前阶段**：MVP 立项（PRD + 架构设计）
- **目标平台**：Android（后续可移植 iOS）
- **技术引擎**：Unity 2022 LTS + C#
- **预计 MVP 周期**：3-4 个月（含调优）

## MVP 范围（v0.2）

- 8 个剧情关卡（晋阳起兵 → 浅水原 → 虎牢关 → 玄武门前夜）
- **15 名可上阵角色**（出阵选 8）：
  - **君主**：李世民
  - **武将（8人）**：李靖、尉迟敬德、秦琼、程咬金、侯君集、段志玄、刘弘基、殷开山
  - **谋士（4人）**：长孙无忌、房玄龄、杜如晦（+长孙皇后兼医师定位）
  - **女性（2人）**：平阳公主、长孙皇后
  - **羁绊**：帝后（李世民+长孙皇后）、夫妻档（柴绍+平阳公主）、瓦岗三杰（秦琼+程咬金+尉迟敬德）
- 5 种兵种（步/重步/骑/弓/器械/谋士）、8 种计策、装备/技能成长体系
- 高度还原曹操传：六边形/方格战棋、兵种克制、计策系统、武将单挑、剧情分支
- 主菜单 → 大地图 → 战斗 → 战后结算 → 存档 完整链路
