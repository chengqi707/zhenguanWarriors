# 贞观勇士（Zhenguan Warriors）

> 一款致敬《曹操传》的回合制战棋 RPG 手游，主人公为李世民，讲述其从晋阳起兵到登基为帝的征战史诗。

## 项目状态

- **当前阶段**：MVP 立项（PRD + 架构设计）
- **目标平台**：Android（后续可移植 iOS）
- **技术引擎**：Unity 2022 LTS + C#
- **预计 MVP 周期**：3-4 个月（含调优）

## MVP 范围（v0.1）

- 8 个剧情关卡（晋阳起兵 → 浅水原 → 虎牢关）
- 12 名可上阵武将（李世民、长孙无忌、尉迟敬德、秦琼、程咬金、李靖、侯君集、段志玄、刘弘基、殷开山、长孙顺德、柴绍）
- 高度还原曹操传：六边形/方格战棋、兵种克制、计策系统、武将单挑、剧情分支
- 装备/技能/经验成长系统
- 主菜单 → 大地图 → 战斗 → 战后结算 → 存档 完整链路
