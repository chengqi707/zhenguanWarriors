# 贞观勇士 开发路线图 v0.1

## MVP 里程碑（共 4 阶段，每阶段约 3-4 周）

### Phase 1：核心战斗骨架（第 1-4 周）
**目标**：跑通 1 关战斗 demo，验证手感

- 创建 Unity 工程、配置 URP 2D、接入 Tiled2Unity
- 六边形网格 + A* 寻路
- 战斗回合控制器、基础攻击/移动/待机
- 1 个简化武将（李世民）+ 2 个敌兵
- 1 张测试地图（霍邑外平原）
- 占位贴图（彩色方块）+ 占位音效

**交付**：可玩 demo APK，能跑通一回合攻防

### Phase 2：完整战斗系统（第 5-8 周）
**目标**：4 关完整玩法，6 角色可上阵（含谋士/女性兵种验证）

- 兵种相克（含新增谋士兵种）、地形效果、命中暴击
- 计策 4 种（火攻/落石/鼓舞/医疗）
- 武将单挑系统
- 装备系统（武器+防具+饰品，含性别/职业限定装备）
- 经验/升级/技能习得
- AI 行为树（基础+计策决策，谋士AI需保持后排）
- 4 关数据（晋阳起兵→霍邑→长安→浅水原）
- 上阵阵容：李世民、李靖、尉迟敬德、秦琼、长孙无忌（谋士）、平阳公主（女性）

**交付**：4 关可通关的 alpha 版

### Phase 3：剧情与系统完善（第 9-12 周）
**目标**：补足 4 关 + 9 角色，剧情对话上线，羁绊系统

- 对话系统（立绘 + 文本 + 选项）
- 关卡事件触发器（回合数/位置/HP）
- 剧情分支（影响后续关卡）
- 其余 4 关（柏壁→洛阳→虎牢关→玄武门前夜）
- 其余 9 角色：程咬金、侯君集、段志玄、刘弘基、殷开山、柴绍、房玄龄（谋士）、杜如晦（谋士）、长孙皇后（女性）
- 完整 30 件装备、25 个技能
- 羁绊系统（帝后/夫妻/瓦岗三杰加成）
- 主菜单/大地图/编组/存档完整链路
- 计策 8 种全部上线（含混乱/洞察）

**交付**：内容完整的 beta 版

### Phase 4：打磨与发布（第 13-16 周）
**目标**：UI/UX 打磨，性能优化，上架

- UI 美化（朱红+宣纸黄主题）
- BGM/SFX 接入（免费素材）
- 角色立绘 15 套（含 2 女性角色，AI 生成或免费素材统一调风）
- 战斗特效（DOTween 动画）
- 性能优化（DC<50、内存<300MB）
- 多分辨率适配
- 内测 10 人，收集反馈调平衡（重点验证谋士存活率、女性角色平衡）
- Release 包打包、签名、上架（TapTap）

**交付**：v1.0 正式版

## Token 预算分配（MVP 累计约 1500-2500 万）

| 阶段 | Token 预算 | 重点消耗 |
|---|---|---|
| Phase 1 | 300-500 万 | 战棋引擎搭建、寻路调试 |
| Phase 2 | 500-800 万 | 战斗AI、平衡性、计策实现 |
| Phase 3 | 400-700 万 | 剧情脚本、关卡数据、对话系统 |
| Phase 4 | 300-500 万 | UI打磨、Bug修复、性能优化 |

## 风险检查点

- **Phase 1 末**：手感是否对？如不对，调整战棋核心（决策点）
- **Phase 2 末**：AI 是否好玩？平衡性如何？
- **Phase 3 末**：内容量是否足够？剧情是否吸引人？
- **Phase 4 末**：性能与稳定性是否达标？

## 后续版本规划（MVP 后）

- v1.1：剧情DLC（玄武门之变 + 贞观之治 5 关）
- v1.2：自由模式（攻略全图、武将自由招募）
- v1.3：联机对战（PVP 战棋对决）
- v2.0：iOS 版本、云存档
