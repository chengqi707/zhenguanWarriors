# 06 UI/UX 设计规范

> 本文档定义全部界面的视觉风格、布局规范、控件标准与交互反馈。目标：**唐风古韵 + 清晰易用 + 单手/双手皆可操作**。

---

## 1. 设计原则

1. **一眼唐风**：朱红+宣纸黄+墨黑，玩家看到颜色就想起大唐。
2. **信息分层**：战场信息分三级——必看（血条/回合）、重要（计策/地形）、辅助（Log/详情）。
3. **触控优先**：所有点击热区 ≥ 80×80dp，避免误触。
4. **即时反馈**：任何操作 100ms 内必须有视觉响应。
5. **可跳过**：动画、剧情、敌方回合——全部可跳过或加速。

## 3. 字体规范

### 3.1 字体选择

| 用途 | 字体 | 备选 |
|---|---|---|
| 标题/剧情 | 思源宋体 | 方正书宋 |
| UI文字/数字 | 思源黑体 | 方正兰亭黑 |
| 数字/血量 | 等宽数字字体 | Roboto Mono |

> Unity 中使用 TextMeshPro，字体通过 Font Asset Creator 生成 Atlas。

### 3.2 字号层级

| 层级 | 用途 | 基准字号 | 字重 |
|---|---|---|---|
| H1 | 关卡标题、主菜单Logo | 48px | Bold |
| H2 | 面板标题、角色名 | 36px | Bold |
| H3 | 按钮文字、标签 | 28px | Medium |
| Body | 正文、剧情对话 | 24px | Regular |
| Caption | 辅助说明、属性名 | 20px | Regular |
| Number | 血量/伤害数字 | 32px | Bold |
| Tiny | Log、版本号 | 16px | Regular |

### 3.3 数字显示

- 血量：`当前/最大`，当前值低于25%时变红闪烁
- 伤害：弹出数字，暴击放大1.5倍+红色，治疗绿色
- 资金：图标+数字，图标为铜钱形状
- 经验：进度条+「XXX/YYY」

## 4. 屏幕适配

### 4.1 基准分辨率

- **设计基准**：1920×1080（横屏16:9）
- **最小适配**：1280×720
- **最大适配**：2400×1080（手机全面屏）
- **纵横比**：16:9 为主，18:9/20:9 自动扩展左右边距

### 4.2 安全区

- 顶部：预留 60px（状态栏/刘海）
- 底部：预留 80px（手势条/导航栏）
- 左右：最小 40px 边距
- 关键按钮避开四角（误触区）

### 4.3 横屏锁定

- MVP 阶段**锁定横屏**，不响应竖屏切换。
- 手机端双手持握，左手移动/选择，右手确认/菜单。

- 左侧：回合数（H2）

- 六边形网格占屏幕 70%

- 显示所有己方单位小头像
- 绿色边框=可行动，灰色=已行动，黑色=阵亡
- 点击头像快速定位到该单位

#### 操作提示栏
- 动态提示当前操作步骤
- 结束回合按钮（大）
- 存档按钮（小）

## 6. 控件规范

### 6.1 按钮

| 状态 | 背景 | 边框 | 文字 | 阴影 |
|---|---|---|---|---|
| 正常 | 宣纸黄 `#F5E6C8` | 朱红 `#C23A30` 2px | 墨黑 | 无 |
| 悬停/按下 | 朱红 `#C23A30` | 无 | 雪白 | 无 |
| 禁用 | 赭石 `#8B5A2B` | 烟灰 `#888888` | 烟灰 | 无 |
| 高亮 | 藤黄 `#F0C040` | 朱红 | 墨黑 | 微发光 |

- 圆角：8px
- 最小尺寸：160×64px
- 文字居中，字重 Medium

### 6.2 面板/弹窗

- 背景：宣纸黄 `#F5E6C8`，透明度 95%
- 边框：朱红 `#C23A30`，2px
- 边角：卷轴风格（上下两边微卷）
- 标题栏：朱红底+白字，左侧竖线装饰
- 关闭按钮：右上角「×」

### 6.3 血条

```
┌────────────────────┐
│███████░░░ 李世民   │  绿色=良好
│ HP: 85/100         │
└────────────────────┘
```

- 背景：墨黑 `#1A1A1A`
- 填充：
  - >50%：石青 `#5B8A72`
  - 25-50%：藤黄 `#F0C040`
  - <25%：朱红 `#C23A30`（闪烁）
- 高度：8px，圆角 4px
- 名字在血条上方或右侧

### 6.4 进度条

- 经验条：蓝紫渐变，细线（4px）
- 回合进度：顶部细条，玩家回合青色，敌方回合红色

## 10. 无障碍

1. **字体大小**：设置内可切换小/中/大三级
2. **色盲模式**：可选色盲滤镜（红绿/蓝黄），单位用形状区分（圆/方/三角）
3. **震动开关**：可全局关闭震动反馈
